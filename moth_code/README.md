# This is a prototype of a moving moth using p5js

This is code that is a part of a bigger installation.

It features *mediapipe handtracking*, to navigate hand landmarks on screen. This hand tracking is overlaying p5js graphics.

There are 3 circles on the screen, which when hovered over by the hand: display information about moth species.

The hand movement may appear slow moving even with a good processor, this is intentional as the full installation has been designed to work on a raspberry pi.
